# Olist E-Commerce Sales Analysis

## Project Overview
This project analyzes the Olist e-commerce dataset to uncover insights into
sales performance, customer behavior, delivery efficiency, payments and reviews
using SQL, Excel, Power BI and Tableau.

## Tools Used
- SQL
- Excel
- Power BI
- Tableau

## Business Objectives
- Analyze sales trends and revenue performance
- Study customer purchasing patterns
- Evaluate order delivery performance
- Understand payment behavior
- Analyze customer reviews and satisfaction

## Key Insights
- Identified top-performing product categories
- Analyzed monthly sales trends
- Evaluated delivery delays and shipping performance
- Studied customer review patterns
- Built interactive dashboards for business insights

## Files Included
- SQL Queries
- Excel Analysis
- Power BI Dashboard
- Tableau Dashboard
